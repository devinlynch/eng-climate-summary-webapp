# Getting Started

## Prerequisites

1. Java must be installed


## Running the Tests

### Windows
1. cd eng-climate-summary-webapp
2. gradlew.bat test

### Linux
1. cd eng-climate-summary-webapp
2. ./gradlew test

## Running the App

### Windows
1. cd eng-climate-summary-webapp
2. gradlew.bat bootRun

### Linux
1. cd eng-climate-summary-webapp
2. ./gradlew bootRun


## Accessing the App
Visit http://127.0.0.1:8080/ in your web browser
